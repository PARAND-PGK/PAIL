# 📨 PAIL — سامانه پیام‌رسانی خصوصی درون تلگرام

**PAIL** یک ربات تلگرامی است که یک سامانه پیام‌رسانی خصوصی و شبیه به ایمیل را شبیه‌سازی می‌کند.
توجه: PAIL یک سرویس ایمیل واقعی اینترنتی نیست و فقط داخل تلگرام کار می‌کند.

## ✨ ویژگی‌ها

- ثبت‌نام و ورود با نام کاربری و رمز عبور (رمزهای عبور با الگوریتم PBKDF2 هش می‌شوند)
- ساخت یک یا چند آدرس یکتا با فرمت `something@pail.com`
- ارسال پیام جدید با موضوع، متن و امکان پیوست کردن عکس، ویدیو، موزیک، ویس، فایل، zip، pdf و هر نوع فایل دیگر
- صندوق ورودی و ارسال‌شده‌ها با قابلیت باز کردن، حذف و ستاره‌دار کردن پیام‌ها
- پروفایل کاربری (نام کاربری، عکس پروفایل تلگرام، بیوگرافی، لیست آدرس‌ها)
- تنظیمات: تغییر رمز عبور، ساخت/حذف آدرس، خروج از حساب
- ذخیره‌سازی پیام‌ها و فایل‌ها در دیتابیس SQLite و پوشه‌های محلی

## 🗂 ساختار پروژه

```
PAIL/
├── config/          # تنظیمات پروژه (خواندن متغیرهای محیطی)
├── database/        # اتصال به دیتابیس و اسکیمای جدول‌ها
├── handlers/         # هندلرهای aiogram (منطق مکالمات)
├── keyboards/        # کیبوردهای ReplyKeyboard و InlineKeyboard
├── middlewares/       # میان‌افزار احراز هویت و فیلترها
├── models/           # گروه‌های وضعیت FSM
├── services/          # منطق تجاری (کاربر، آدرس، پیام، فایل)
├── utils/            # توابع کمکی (اعتبارسنجی، امنیت، قالب‌بندی)
├── storage/attachments/  # محل ذخیره فایل‌های پیوست
├── main.py           # نقطه ورود اصلی برنامه
├── requirements.txt
├── .env.example
└── README.md
```

## ⚙️ نصب

1. پایتون ۳.۱۲ یا بالاتر نصب باشد.
2. یک محیط مجازی بسازید (اختیاری ولی پیشنهادی):

```bash
python -m venv venv
source venv/bin/activate      # در ویندوز: venv\Scripts\activate
```

3. وابستگی‌ها را نصب کنید:

```bash
pip install -r requirements.txt
```

## 🔑 قرار دادن توکن ربات (BOT TOKEN)

1. فایل `.env.example` را کپی کرده و نام آن را به `.env` تغییر دهید:

```bash
cp .env.example .env
```

2. فایل `.env` را باز کنید و توکن ربات تلگرام خود را (که از [@BotFather](https://t.me/BotFather) گرفته‌اید) وارد کنید:

```
BOT_TOKEN=YOUR_TOKEN
```

مثال واقعی:

```
BOT_TOKEN=123456789:AAExampleTokenNotReal1234567890
```

می‌توانید مسیر دیتابیس و پوشه ذخیره‌سازی را هم در همان فایل تغییر دهید (اختیاری):

```
DB_PATH=database/pail.db
STORAGE_PATH=storage/attachments
```

## ▶️ اجرا

```bash
python main.py
```

اگر همه چیز درست باشد، در ترمینال پیام «ربات PAIL در حال اجراست...» را می‌بینید.
حالا می‌توانید در تلگرام به ربات خود پیام `/start` بدهید و از PAIL استفاده کنید.

## 🔒 امنیت

- رمزهای عبور هرگز به صورت متن ساده ذخیره نمی‌شوند؛ با الگوریتم PBKDF2-HMAC-SHA256 و salt تصادفی هش می‌شوند.
- تمام ورودی‌های کاربر (نام کاربری، رمز عبور، آدرس) اعتبارسنجی می‌شوند.
- آدرس‌های PAIL باید یکتا باشند و هنگام ساخت بررسی می‌شوند.

## 📝 نکات

- همه پیام‌های ربات به زبان فارسی هستند؛ آدرس‌های ایمیل (مانند `ali@pail.com`) به انگلیسی باقی می‌مانند.
- فایل دیتابیس SQLite به صورت خودکار در مسیر تعیین‌شده (`DB_PATH`) ساخته می‌شود.
- فایل‌های پیوست در `storage/attachments/<message_id>/` ذخیره می‌شوند.
